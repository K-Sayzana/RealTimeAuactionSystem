# 🏆 Real-Time Online Auction Platform

A modern web-based auction system that enables sellers to list items and bidders to participate in live auctions in real time. The platform provides a seamless bidding experience with instant updates, interactive communication features, and efficient auction management.

## ✨ Features

### 📦 Auction Item Management

* Create and manage auction listings with item details and starting prices.
* Configure auction durations and monitor active listings.
* Browse a catalog of available auction items across multiple categories.

### ⚡ Real-Time Bidding

* Place bids on active auctions with immediate feedback.
* Instantly update the current highest bid across connected users.
* Track auction progress and bidding activity in real time.
* Prevent invalid or malformed bid submissions through built-in validation.

### 💬 Live Interaction

* Integrated chat system for communication between participants.
* Real-time activity feed displaying bids, auction events, and system notifications.
* Immediate updates without requiring page refreshes.

### 🔐 User Session Management

* Support for multiple concurrent user sessions.
* Isolated account testing across browser tabs.
* Secure logout functionality with automatic cleanup of cached session data.

## 🏗️ Technology Stack

### Backend

* Java 17+
* Spark Java
* Gson / Jackson

### Frontend

* HTML5
* CSS3
* Vanilla JavaScript (ES6+)

### Real-Time Communication

* WebSockets / Server-Sent Events (SSE)

### Prerequisites

Make sure the following tools are installed:

* Java Development Kit (JDK) 17 or higher
* Maven
